## Dependencies
The application requires Python2.7 and pip installed. All the rest dependencies are indicated in requirements.txt

## How to run it
You can execute
  - bash run.sh
  - run.bat
  - or next commands:
```
pip install -r requirements.txt
cd src
python2.7 -m main.py
```
